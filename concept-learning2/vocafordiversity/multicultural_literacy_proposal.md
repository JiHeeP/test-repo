# 다문화 문해력 사다리
학습도구어 기반 5단계 어휘 학습 플랫폼 제안서 (개발 실행본)

## 1. 문서 목적
이 문서는 다음을 한 번에 정의한다.
- 왜 이 제품을 만드는지 (교육적 목표)
- 무엇을 1차 출시(2026-03-01)까지 구현해야 하는지 (제품 범위)

이 문서는 기획서이자 개발 기준 문서다.

---

## 2. 문제 정의
초등 저학년 교실에서 다문화 배경 학생은 학습도구어(Tier 2) 노출량이 부족해 다음 영역에서 불리해진다.
- 교과 지시문 이해
- 개념 설명/비교/분류 같은 학습 언어 산출
- 문장 구성과 조사 선택 정확도

핵심 문제는 단순 단어 암기가 아니라, 교과 맥락에서 단어를 "산출"하지 못하는 데 있다.

---

## 3. 제품 목표
### 3.1 교육 목표
- 학습도구어를 수용 수준에서 산출 수준으로 끌어올린다.
- 학생별 오류율을 추적해 Tier 2/Tier 3 개입 대상을 자동 식별한다.
- 다문화 학생과 전체 학생 간 단계별 격차를 줄인다.

### 3.2 운영 목표
- 교사는 수업 중 실시간으로 위험 학생을 확인할 수 있어야 한다.
- 학생 리포트와 월간 그룹 리포트를 PDF로 바로 출력할 수 있어야 한다.
- 개입 이력(무엇을, 언제, 얼마나, 효과가 있었는지)을 데이터로 남긴다.

---

## 4. 이론-기능 매핑
### 4.1 Nation(2013) 매핑
- Level 1~2: 재인 중심
- Level 3: 의미 관계 판단
- Level 4: Meaning-Productive (Controlled Production)
- Level 5: Use-Productive (Controlled Production)

주의:
- Level 4/5는 자유 산출(Free Production)이 아니라 통제 산출(Controlled Production)으로 정의한다.

### 4.2 RTI 매핑
- Tier 1: 전체 학생 공통 학습
- Tier 2: 소그룹 보충
- Tier 3: 1:1 집중 지도

### 4.3 오류율 판정(확정)
- 0 <= 오류율 <= 20: 습득 완료
- 20 < 오류율 <= 35: 발달 중
- 35 < 오류율 <= 50: Tier 2
- 50 < 오류율: Tier 3

## 6. 학습 경험 설계
### 6.1 세션 규칙
- 1회 학습 목표 시간: 10~20분
- 1차에서는 시간 제한 없음
- 힌트 1회
- 재시도 1회
- 단계 자동 이동
- 문항 순서 랜덤

### 6.2 단계 정의
- Level 1 암기카드: 단어-뜻-그림 노출
- Level 2 매칭게임: 단어-의미 연결 강화
- Level 3 속성판단: 의미/관계 판단
- Level 4 음절 선택 완성: 빈칸 단어 완성
- Level 5 어절 조립: 문맥/문법을 고려한 문장 완성

### 6.3 Level 4/5 채점 단위
- 정답: 2점
- 힌트 후 정답: 1점
- 오답/재시도 실패: 0점

### 6.4 level 1 그림 삽입(요청에 따른 삽입)
- lucid-react 에서 icon과 단어 목록 매칭
