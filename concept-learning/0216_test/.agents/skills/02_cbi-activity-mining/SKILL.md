# 목표
매일 1:10-2:10(활동 채굴 60분) 구간에서, 오늘 읽은 방법/전략을 **활동 카드 1장**으로 고정 산출물화한다.

# 입력(필수)
- 오늘의 근거 문장/설계 규칙(없으면 “없음”이라고 명시)
- 오늘 읽은 내용에서 가져올 활동/전략 후보 1~2개(제목만)

# 진행(티키타카)
1) 먼저 사용자가 후보를 1~2개 적게 하고, **최종 1개만** 고르게 한다(욕심 금지).
2) 최종 1개를 고르면, 아래를 한 번에 채우게 한다.
   - 활동의 핵심 기능(1줄): (예: 근거 제시, 비교, 분류, 관점 전환, 개념 정교화)
   - 활동의 타깃 사고(1줄): 학생이 무엇을 “생각하게” 만드는가?
   - 사용 조건/주의(1줄): 언제 쓰면 망가지나?
3) 이어서 `daily_practice_until_grade_confirmed.md`의 **활동 카드 템플릿 전 항목**을 작성하게 한다.
4) 마지막에 정렬 문장 1줄을 강제한다.
   - “이 활동은 ________을(를) 강화하기 위해 ________을 요구한다.”

# 품질 체크(통과 못 하면 고쳐쓰기)
- 언어 스캐폴딩이 정확히 2개인가?
- 출력(산출물)이 1개로 고정돼 있는가?
- 남길 근거 2개가 “산출물 1 + 피드백 흔적 1”로 써 있는가?
- 정렬 문장이 CBI(개념/일반화/탐구질문) 중 무엇을 강화하는지 분명한가?

# 출력(고정 포맷, 코드블록 금지)
- 활동 카드(템플릿):
  - 활동명:
  - CBI 연결:
  - 학생 행동:
  - 교사 행동:
  - 언어 스캐폴딩 2개:
  - 입력 텍스트:
  - 출력(산출물):
  - 남길 근거 2개:
- 정렬 문장(1줄):

# 3회 시도 규칙
- 같은 요청이 3회 반복되면, “활동 카드”는 사용자가 채우게 두고, **정렬 문장 예시 1줄만** 제공한다.
